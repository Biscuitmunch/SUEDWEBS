# SUEDWEBS

Ensure you have [node](https://nodejs.org/) installed.

Create a file `terraria.db` in `/server/src/db`

Terminal 1

```
cd client
cp .env.template .env
npm i
npm run dev
```

Terminal 2

```
cd server
npm i
npm run dev
```

Ensure the .env file is pointing to the port that is running in server.
